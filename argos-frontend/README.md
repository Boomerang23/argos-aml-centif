# ARGOS AML Frontend

Frontend premium de la plateforme ARGOS AML Immobilier CI.

## Stack
- Next.js
- TypeScript
- Tailwind CSS
- shadcn/ui
- TanStack Query

## Fonctionnalités couvertes
- Login
- Dashboard AML
- Liste des dossiers
- Création dossier
- Détail dossier
- Parties
- KYC / onboarding
- Screening
- Décision conformité
- Audit trail
- Export PDF
- Administration utilisateurs
